Plataforma de Jogos

Aplicação web interativa com múltiplos jogos, sistema de ranking, chat em tempo real e perfis de usuários personalizados.

📱 Ver no GitHub 🌐 Demo Online

Este projeto é uma plataforma de jogos online que oferece uma experiência de usuário rica e interativa. Os jogadores podem competir em tempo real, conversar com outros usuários e acompanhar seu progresso em um ranking global.

🎯 Principais Funcionalidades

  • 🎲 Múltiplos jogos implementados com Canvas API.
  • 💬 Chat em tempo real com salas e mensagens privadas usando WebSockets.
  • 🏆 Sistema de ranking global e por jogo.
  • 👤 Perfis de usuário personalizados com estatísticas de jogo.
  • 🔥 Autenticação e armazenamento de dados com Firebase.
