## 🚀 **Como Usar**

### **Navegação**
1. **Desktop**: Use o menu superior
2. **Mobile**: Toque no ícone hamburger (☰)
3. **Teclado**: Tab para navegar, Enter para selecionar

### **Funcionalidades**
- 🔍 **Buscar no blog**: Digite na caixa de busca
- 🏷️ **Filtrar posts**: Clique nos botões de categoria
- 📧 **Contato**: Preencha o formulário na página de contato
- 🖼️ **Ver imagens**: Clique nas imagens da galeria
- 🌙 **Trocar tema**: Clique no botão "Tema" (ainda em desenvolvimento)

### **Páginas Principais**
- 🏠 **Início**: Portfolio principal com projetos

- 📞 **Contato**: Formulário e informações
- 💼 **Projetos**: Detalhes de cada projeto
